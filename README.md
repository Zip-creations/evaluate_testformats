# evaluate_testformats

Eine Evaluation benutzter Testausbageformate ausgewählter Testframeworks aus einer Auswahl von etablierten Programmiersprachen. 

Standartmäßig befinden sich die Testdateien unter \<sprache\>/src/test/\<framework\>, und die produzierten Outputs unter \<sprache\>/src/out/\<framework\>.<br>
Es existieren einige Ausnahmen, da manche Testframeworks eine angepasste Struktur und/oder Namenskonventionen vorraussetzen.

Dieser Datensatz wurde unter Ubuntu 24.04.4 LTS in Kombination mit dem package manager nix erstellt.
